# Deploy your schematics with ease

Create: Buzzy Bees adds a few handy features to create, allowing you to build your schematics with ease.

# Notes about 1.20.1 Support

There is an experimental branch based of the version 1.1.2:
https://github.com/StaticFX/Create-Buzzy-Bees/tree/feature/multi-version

Jar Filed are published to curseforge and modrinth as well

## Futute of 1.20.1 Support

Support for 1.20.1 exists only for bugfixes but will not receive any new features. 1.20.1 does not only differ in the
version, but also in the modloader as it needs to depend on forge which adds in a lot of complexity. Also maintaining 2
versions, developing in 2 versions at the same time, consumes more time. Im developing this mod out of fun, and modding
in 2 version in parallel is not a lot of fun.

## Features

- Bees & Beehives: Have bees build or destroy schematics around your world
- Bumble Bees: Transport items from one cargo port to another
- Construction Planner: Easily manage your schematics in one place
- Portable Beehive: Move your bees around with you
